# Hotel Management System

A hotel management system built with MySQL and Python.

## Technologies
- MySQL Workbench 8.4
- Python 3.13
- mysql-connector-python

## Features
- Guest Management
- Room Management
- Service Management
- Booking Management
- Invoice Management
- Reports & Views

## How to Run
1. Import `hotel.sql` into MySQL Workbench
2. Install dependencies: `pip install mysql-connector-python`
3. Update password in `db.py`
4. Run: `python main.py`
